# EduLearn - Educational Platform

A modern educational website built with HTML, CSS, and JavaScript, featuring course management, user authentication, and an admin panel.

## Features

- **Course Listings**: Browse available courses
- **User Authentication**: Login and register functionality
- **Admin Panel**: Add and delete courses (admin access required)
- **Responsive Design**: Works on desktop and mobile devices
- **Modern UI/UX**: Clean and intuitive interface

## Getting Started

1. Clone or download the project files
2. Open `index.html` in your web browser
3. For admin access, use:
   - Email: admin@edulearn.com
   - Password: admin123

## File Structure

```

├── index.html          # Home page
```

## Usage

### For Students
- Register a new account or login
- Browse available courses on the home page

### For Admins
- Login with admin credentials
- Access the admin panel via `/admin/index.html`
- Add new courses with title, description, instructor, and price
- Delete existing courses

## Deployment

This is a static website that can be deployed to any web hosting service:

- **GitHub Pages**: Push to a GitHub repository and enable Pages
- **Netlify**: Drag and drop the folder or connect to a Git repository
- **Vercel**: Similar to Netlify, supports static sites
- **Traditional hosting**: Upload files via FTP
